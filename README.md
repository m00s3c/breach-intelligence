## Breach Lookup & Monitoring

- [Have I Been Pwned](https://haveibeenpwned.com/) - Check if your email has been compromised in data breaches
- [DeHashed](https://dehashed.com/) - Paid breach search engine with extensive dataset
- [IntelligenceX](https://intelx.io/) - OSINT search engine including breach data
- [LeakCheck](https://leakcheck.io/) - Email breach lookup and monitoring
- [BreachDirectory](https://breachdirectory.org/) - Free data breach search
- [Snusbase](https://snusbase.com/) - Breach database search
- [Ghost Project](https://ghostproject.fr/) - Password breach checker

## Dark Web Monitoring

- [DarkOwl](https://www.darkowl.com/) - Dark web data collection and monitoring
- [Flashpoint](https://www.flashpoint-intel.com/) - Threat intelligence from dark web sources
- [Recorded Future](https://www.recordedfuture.com/) - Real-time threat intelligence
- [Digital Shadows](https://www.digitalshadows.com/) - Digital risk protection
- [Flare](https://flare.io/) - Threat exposure monitoring
- [ZeroFox](https://www.zerofox.com/) - External cybersecurity platform
- [Webhose.io](https://webhose.io/) - Dark web data feeds
- [SpyCloud](https://spycloud.com/) - Account takeover prevention

## OSINT Tools

- [Maltego](https://www.maltego.com/) - Data mining and link analysis
- [Shodan](https://www.shodan.io/) - Search engine for Internet-connected devices
- [Censys](https://censys.io/) - Internet-wide scanning and search
- [Recon-ng](https://github.com/lanmaster53/recon-ng) - Web reconnaissance framework
- [theHarvester](https://github.com/laramies/theHarvester) - E-mail, subdomain and people names harvester
- [SpiderFoot](https://www.spiderfoot.net/) - Automated OSINT tool
- [Hunchly](https://www.hunch.ly/) - Web investigation tool
- [OSINT Framework](https://osintframework.com/) - Collection of OSINT tools

## Threat Intelligence Platforms

- [AlienVault OTX](https://otx.alienvault.com/) - Open threat intelligence community
- [MISP](https://www.misp-project.org/) - Threat intelligence sharing platform
- [ThreatConnect](https://threatconnect.com/) - Threat intelligence operations
- [Anomali](https://www.anomali.com/) - Threat intelligence and security operations
- [ThreatQuotient](https://www.threatq.com/) - Threat intelligence platform
- [RiskIQ](https://www.riskiq.com/) - Digital threat management

## Password Security

- [1Password Watchtower](https://watchtower.1password.com/) - Monitor password breaches
- [Bitwarden Breach Report](https://bitwarden.com/) - Password manager with breach alerts
- [KeePass](https://keepass.info/) - Open-source password manager
- [Password Checkup](https://passwords.google.com/checkup) - Google password breach checker

## Data Leak Search Engines

- [Pwndb](https://github.com/davidtavarez/pwndb) - Leaked credentials search
- [WeLeakInfo](https://weleakinfo.to/) - Data breach search (mirror sites)
- [Dehashed API](https://www.dehashed.com/docs) - Programmatic breach search
- [SecurityTrails](https://securitytrails.com/) - Historical DNS and WHOIS data
